# Campaign Sequence 09: Post-Login Orientation

## Automation purpose

Help a member understand how to use Positives after first login.

## Trigger

- `first_login_complete`

## Delivery lane

`Standard ActiveCampaign`

## Email 1: You’re In

- Timing:
  - immediately after first login
- Subject:
  - `You’re in — here’s where to start`
- Preview:
  - `A simple way to use Positives well from day one.`
- CTA:
  - `Go To Today`
- CTA link:
  - `https://positives.life/today`

### Draft copy

Hi %FIRSTNAME%,

You’re in.

The simplest way to use Positives is this:

- start with Today
- stay close to This Week
- let This Month give your practice a larger direction

You do not need to do everything. You just need to return.

Button:
`Go To Today`

## Email 2: The Best Way To Use Positives

- Timing:
  - 2 days later
- Subject:
  - `The easiest way to get value from Positives`
- Preview:
  - `Keep it simple: return daily, not perfectly.`
- CTA:
  - `Open Positives`
- CTA link:
  - `https://positives.life/today`

### Draft copy

Hi %FIRSTNAME%,

The best way to get value from Positives is not to do more. It’s to come back
more consistently.

Even a few quiet minutes with today’s practice can reset the tone of your day.

If you miss a day, just return the next day. Positives is a practice, not a
program to complete.

Button:
`Open Positives`
